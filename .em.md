# Rust TIOBE Deploy Project

## 项目简介
这是一个基于 Rust 和 Axum 框架开发的 Web 应用程序，旨在展示和查询 TIOBE 编程语言排行榜数据。

主要功能包括：
- **实时数据获取**：从 TIOBE 官方网站抓取最新的编程语言排名数据。
- **历史数据查询**：支持通过年份和月份查询历史排名数据。
- **语言详情展示**：提供特定编程语言的详细信息，包括描述、主要用途和流行框架。
- **RESTful API**：提供 JSON 格式的 API 接口，方便前端或其他客户端调用。
- **静态资源服务**：内置静态文件服务器，用于托管前端可视化界面。

## 技术栈
- **后端框架**: Axum
- **HTTP 客户端**: Reqwest
- **HTML 解析**: Scraper
- **异步运行时**: Tokio
- **序列化/反序列化**: Serde
- **API 文档**: Utoipa (Swagger UI)

## 快速开始

### 启动服务
确保你已经安装了 Rust 环境，然后在项目根目录下运行：

```bash
cargo run
```

### 访问 Swagger UI 文档
服务启动后，你可以在浏览器中访问以下地址来查看和测试 API 文档：

[http://localhost:3000/swagger-ui/](http://localhost:3000/swagger-ui/)

在 Swagger UI 界面中，你可以：
1. 查看所有可用的 API 端点。
2. 查看每个端点的请求参数和响应结构。
3. 点击 "Try it out" 按钮直接在浏览器中发送请求并查看结果。

![Swagger UI 示例](https://raw.githubusercontent.com/swagger-api/swagger-ui/master/docs/images/swagger-ui-screenshot.png)
*(注：上图为 Swagger UI 官方示例截图，实际界面会根据本项目 API 定义显示)*

### 访问主页
前端可视化界面可以通过以下地址访问：
