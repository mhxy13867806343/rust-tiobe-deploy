# Rust TIOBE Deploy Project

## 项目简介
这是一个基于 Rust 和 Axum 框架开发的 Web 应用程序，旨在展示和查询 TIOBE 编程语言排行榜数据。

主要功能包括：
- **实时数据获取**：从 TIOBE 官方网站抓取最新的编程语言排名数据。
- **历史数据查询**：支持通过年份和月份查询历史排名数据。
- **语言详情展示**：提供特定编程语言的详细信息，包括描述、主要用途和流行框架。
- **RESTful API**：提供 JSON 格式的 API 接口，方便前端或其他客户端调用。
- **静态资源服务**：内置静态文件服务器，用于托管前端可视化界面。

## 技术栈
- **后端框架**: Axum
- **HTTP 客户端**: Reqwest
- **HTML 解析**: Scraper
- **异步运行时**: Tokio
- **序列化/反序列化**: Serde
- **API 文档**: Utoipa (Swagger UI)
